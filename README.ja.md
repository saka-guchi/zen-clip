**ZenClip** は、Windowsのタスクトレイに常駐するAI搭載のクリップボードマネージャーです。
ローカル環境にインストールされた **Gemini CLI** と連携し、作業の手を止めることなく、クリップボードのテキストをカスタムプロンプトで瞬時に処理します。

**基本無料（機能制限あり）** でご利用いただけます。PolarでProライセンスを購入いただくことで、すべての機能が解放されます。

> **Note**: このリポジトリは**リリースファイルの配布**および**バグ報告（Issue）の受付**専用です。ソースコードは非公開となります。

## ⚠️ 前提条件 (必須)

本アプリを利用するには、お使いのPCに **Gemini CLI** がインストールされており、コマンドプロンプト（CMD）から `gemini` コマンドが実行できる状態になっている必要があります。
*事前にGemini CLIのセットアップを完了させてからご利用ください。*

---

## ✨ 主な機能

- **瞬時のAI処理**: テキストをコピーしてグローバルホットキー（デフォルト: `Ctrl + Shift + C`）を押すだけで、AIがバックグラウンドで即座に処理を実行します。
- **邪魔にならない設計**: タスクトレイに静かに常駐。作業の妨げになる大きなウィンドウは開きません。
- **カスタムプロンプト**: トレイアイコンの右クリックメニューから、用途に合わせたAIへの指示（プロンプト）を簡単に切り替えられます。
- **履歴管理**: AIの応答を自動保存。履歴ウィンドウから過去の結果を確認・コピーしたり、複数選択して一括削除することが可能です。
- **ポータブル＆軽量**: 単一の実行ファイル（`.exe`）で動作。複雑なインストール作業は不要です。
- **多言語対応**: UIは英語と日本語に対応（OSの言語設定に合わせて自動で切り替わります）。

### 💎 無料版とPro版（有償ライセンス）の違い
ライセンスキーを入力しない場合でも無料版としてお使いいただけますが、以下の制限があります。

- **無料版**: 履歴の保存件数は〇〇件まで、カスタムプロンプトの登録は〇〇個までとなります。
- **Pro版**: 制限が解除され、すべての機能を無制限にご利用いただけます。

## 🚀 使い方

### 1. インストール
1. **[最新のリリースページ](https://github.com/saka-guchi/zen-clip/releases/latest)** にアクセスします。
2. `ZenClip-vX.Y.Z-win-x64.zip` をダウンロードします。
3. 任意のフォルダにZIPファイルを展開します。
4. `ZenClip.exe` をダブルクリックして起動します。

### 2. 初期設定とアクティベーション
1. 設定 (Settings) 画面から、Polarで取得した **ライセンスキー** を入力してPro版を有効化します（未入力のままでも無料版として利用可能です）。
2. 必要に応じて、プロンプトやグローバルホットキーをカスタマイズしてください。
3. （オプション）AIの処理結果を直接クリップボードへ上書きしたい場合は、設定から「自動コピー」を有効にしてください（デフォルトはOFFです）。
